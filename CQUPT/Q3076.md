Q3076.(语言: C)程序改错-2
从键盘任意输入一个正整数，编程判断它是否是素数，若是素数，输出“Yes!”
，否则输出“No！”。已知负数、0和1都不是素数。请找出
下面程序的错误并改正之，同时按照给出的运行示例检查修改后的程序。

#include <stdio.h>
#include <math.h>
int main()
{
    int n, i;
    printf("Input n:\n");
    scanf("%d", &n);
    for (i = 2; i <= sqrt(n); i++) //sqrt()为开平方根的函数，以此来减小运算。
    {
        if (n % i = 0)
        {
            printf("No!\n");
        }
    }
    printf("Yes!\n");
    return 0;
}
