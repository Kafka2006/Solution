Q1126.(语言: C)有关输入输出问题。
输入为：12a↙
       b↙ 
运行结果为：1,2,a,b,123.300000,65535
请改正程序中的错误，使它能得出正确的结果：
```
#include <stdio.h>

main()
{
    int b;
    unsigned short a = 65535;
    short k = a;
    char c, d;
    int f, g;

    b = (1234.0 - 1) / 10;
    scanf("%c", &c);
    scanf("%c", &d);
    scanf("%d", &f);
    scanf("%d", &g);
    printf("%c,%c,%c,%c,%f,%d", c, d , f, g, b, k);
}
```

#include <stdio.h>

int main()

    double b;

}
